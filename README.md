### Project Description
This repository contains laboratory exercises and firmware developed during the 2026 IoT Summer School program. The coursework focuses on designing smart, interconnected embedded devices with an emphasis on biomedical data acquisition, sensor signal processing, and long-range telemetry. The final project integrates these concepts to implement a prototype telemetry system tailored for medical monitoring and drone-assisted field deployments.

## 📂 Repository Structure
* `/week1` - Introduction to GPIOs and basic actuator control (LED blink architecture).
* `/week2` - Sensor interfacing, analog-to-digital conversion, and biomedical signal filtering.
* `/week3` - Wireless communication protocols and low-power telemetry networks.
* `/final-project` - Integrated firmware codebase for the biomedical subsystem.
